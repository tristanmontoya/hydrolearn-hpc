# HydroLearn HPC Module Activity

This repository contains data and scripts related to the *High-Performance Parallel Computing for Hydrology* module on [HydroLearn](https://hydrolearn.org/). The activity focuses on running the SUMMA hydrological model for the Bow River at Banff, Alberta, Canada. Much of the content in this repository is based on the original SUMMA parameter estimation workflows provided at https://github.com/CH-Earth/summa_calib, adapted for use in the HydroLearn HPC course and execution on the [virtual HPC cluster](https://github.com/tristanmontoya/vhpc-hydrotools).
